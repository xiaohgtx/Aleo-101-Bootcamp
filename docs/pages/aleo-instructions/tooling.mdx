# Aleo Instructions 工具

如果你已经安装了 Leo 语法高亮插件，通常也可以看到 `.aleo` Aleo Instructions 文件的语法高亮。

Aleo 为多个常用编辑器维护了语法高亮支持。如果你常用的编辑器不在列表中，可以到 GitHub 反馈。

当前文档列出的编辑器包括：

1. Sublime Text
2. Visual Studio Code
3. IntelliJ

## Sublime Text

编辑器下载地址：[https://www.sublimetext.com/download](https://www.sublimetext.com/download)。

Sublime 的 Aleo Instructions 支持通过 language server 提供。

### 安装

1. 通过 Package Control 安装 [LSP](https://packagecontrol.io/packages/LSP) 和 [LSP-leo](https://packagecontrol.io/packages/LSP-leo)。
2. 重启 Sublime。

### 使用

按照以下步骤启用 `Aleo instructions` 语法高亮：

1. 打开 `Sublime Text`。
2. 从 `Settings > Select Color Scheme... > LSP-leo` 选择配色方案。
3. 之后你就可以看到 Aleo Instructions 的语法高亮。

## VSCode

编辑器下载地址：[https://code.visualstudio.com/download](https://code.visualstudio.com/download)。

### 安装

1. 从 VSCode Marketplace 安装 [Leo for VSCode](https://marketplace.visualstudio.com/items?itemName=provablehq.leo-extension)。
2. 正确的扩展 ID 是 `provablehq.leo-extension`，描述中应说明它是官方 Leo VSCode 扩展。

### 使用

1. 打开 `VSCode`。
2. 进入 `Settings > Extensions`，或使用左侧面板中的 Extensions 按钮启用 Leo 插件。
3. 之后你就可以看到 Aleo Instructions 的语法高亮。

## IntelliJ

编辑器下载地址：[https://www.jetbrains.com/idea/download/](https://www.jetbrains.com/idea/download/)。

### 安装

1. 从 JetBrains Marketplace 下载 [Aleo Developer Plugin](https://plugins.jetbrains.com/plugin/19858-aleo-developer)。
2. 点击右上角齿轮图标，进入 `Plugins > 齿轮图标 > Install Plugin from Disk`，然后选择下载的 zip 文件。
3. 之后你就可以看到 Aleo Instructions 的语法高亮。

## 选择建议

- 如果你已经使用 VSCode，优先安装官方 Leo 扩展。
- 如果你使用 Sublime Text，可以通过 LSP 和 LSP-leo 获得语法高亮。
- 如果你使用 IntelliJ，可以安装 Aleo Developer Plugin。
